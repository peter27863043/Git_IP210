Boot Loader Upgrade by TFTP
There are two method to do the Boot Loader Upgrade:
1. (By Web)Please browse to or type in the target image file in the upper input field, and then press update button to continue.
2. (By TFTP client)Use MS Windows' Command Prompt window to run tftp client program.
Syntax: c:\tftp -i [CGI_FUF_IP1].[CGI_FUF_IP2].[CGI_FUF_IP3].[CGI_FUF_IP4] put FILE_DIRECTORY\FILENAME.bin
3. If the update process somehow goes wrong(like power failure), please connect to http://[CGI_FUF_IP1].[CGI_FUF_IP2].[CGI_FUF_IP3].[CGI_FUF_IP4] to restart.(If possible, reset device first.)
4. It takes about 30 seconds to complete the Boot Loader Upgrade.


 You'd better carefully read the document regarding the update procedure, preventing the unexpected problem form occurring.
